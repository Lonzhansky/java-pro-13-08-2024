Основи HTML, CSS, JS.

HTML

HTML (HyperText Markup Language) – мова гіпертекстової розмітки.

HTML-документ — файл, що містить HTML. HTML базується на тегах (tag) (прихованих ключових словах), які надають інструкції щодо форматування HTML-документа.

Теги розмітки є фундаментальною характеристикою HTML. Кожен тег розмітки складається з ключового слова, оточеного кутовими дужками, наприклад <html>, <head>, <body>, <title>, <p> тощо.

Теги мають наступний синтаксис:

<tag_name [possible parameters]>

HTML пишеться у вигляді HTML-елементів, що складаються з тегів розмітки. HTML-елемент визначається початковим тегом, певним вмістом та кінцевим тегом. Елементи також можуть містити атрибути, які визначають їх додаткові властивості.

<p>Some text.</p>
<p id="firstName">Bob</p>
<p id="lastName">Green</p>
<a href="https://www.abc.com/">ABC site</a>

<p>, <a> - відкриваючі (початкові) теги.

</p>, </a> - закриваючі (кінцеві) теги.

Some text. , Bob , Green, ABC site - вміст певних елементів.

id, href - атрибути HTML-елементу.

firstName , lastName, https://www.abc.com/ - значення атрибутів HTML-елементів.

Специфікацією HTML передбачається, що HTML-документ (Web-сторінка), структурно ділиться на дві частини: head (голову) та body (тіло).

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
    </head>
    <body>    
        <p>Some text.</p>
    </body>
</html>

<!DOCTYPE html> є оголошенням типу документа. Воно повідомляє Web-браузеру, що цей документ є HTML-документом.

<html> є кореневим елементом HTML-сторінки.

<head> (голова) є елеменом-контейнером для тегів, який надає інформацію про HTML-документ, наприклад, тег <title> визначає назву документа. В голові HTML-документа вказується інформація про документ, наприклад, назва, коротка інструкція, відомості про автора тощо. Вміст цього розділу HTML-файлу не відображається Web-браузером.

<body> (тіло) є елементом, який містить фактичний вміст документа (абзаци, посилання, зображення, таблиці тощо), який відображається у Web-браузері.

HTML-елементи можуть містити інші HTML-елементи. Розміщення одного елемента всередині іншого називається вкладенням. Вкладений елемент є дочірнім відносно батьківського елементу.

<p>My super <b>bold</b> text.</p>
<p><strong>This is my text.</strong></p>

HTML-елементи можна поділити на дві групи: елементи рівня блоку (block-level elements) та елементи вбудованого рівня (inline-level elements). Перші складають структуру HTML-документа, а другі впорядковують вміст блоку.

Елементи рівня блоку візуально відформатовані як блоки, займають повну доступну ширину. Найпоширенішими елементами на рівні блоку є: <div>, <p>, <h1> - <h6>, <form>, <ol>, <ul>, <li>.

Елементи вбудованого рівня не утворюють нових блоків вмісту. Часто використовувані вбудовані елементи: <img>, <a>, <span>, <b>,<i>, <input>, <button>.

CSS

CSS (Cascading Style Sheets) – каскадні таблиці стилів. Простий механізм додавання стилів (шрифтів, кольорів тощо) до Web-документів.

CSS дозволяє спростити процес надання Web-сторінкам презентабельного вигляду, застосовувати стилі на Web-сторінках. Це можна робити незалежно від HTML, з якого складається кожна Web-сторінка.

Сторінки завантажуються швидше під час використання CSS. Не потрібно щоразу писати атрибути HTML-тегів. Просто напишіть одне правило CSS для тега і застосуйте його до всіх входів цього тегу. Таким чином, менше коду – швидший час завантаження.

CSS складається з правил стилю, які інтерпретуються браузером, а потім застосовуються до відповідних елементів у HTML-документі.

Правило стилю складається із трьох частин:

  • Селектор (Selector).
  • Властивість (Property).
  • Значення (Value).

Селектор - HTML-тег, до якого застосовуватиметься стиль. Це може бути будь-який тег, наприклад <h1> або <table> .

Властивість – тип атрибуту HTML-тегу. Усі атрибути HTML перетворюються на властивості CSS. Це може бути колір, межа тощо.

Значення – значення присвоюються властивостям. Наприклад, властивість color може мати значення red або #F1F1F1.

Синтаксис CSS

selector { property: value }

Наприклад,

p {
   color: green;
   font-size: 18px;
}
  • селектор p - абзац.
  • властивість color (колір тексту) має значення green.
  • властивість font-size (розмір шрифту) має значення 18px.

Типи CSS:

  • Вбудований (Inline) містить властивість CSS у розділі body, прикріпленому до елемента.
  • Внутрішній (Internal/Embedded) – набір правил CSS має бути у файлі HTML, у розділі head.
  • Зовнішній (External) – окремий CSS файл, що містить лише властивості стилю за допомогою атрибутів тегів.

JavaScript (JS)

JavaScript (JS) - інтерпретована мова програмування з об’єктно-орієнтованими можливостями.

JavaScript спочатку був відомий як LiveScript, але назва була змінена на JavaScript, можливо через вплив популярності Java. Специфікація ECMA-262 визначає стандартну версію основної мови JavaScript.

Мова JavaScript не має жодного відношення до мови Java.

Спочатку JavaScript був створений, щоб зробити Web-сторінки “живими”.

Програми мовою JavaScript називаються скриптами. Вони можуть вбудовуватися в HTML та виконуватися автоматично під час завантаження Web-сторінки.

JavaScript - це мова сценаріїв, яка використовується для створення та керування динамічним вмістом Web-сайту, тобто всім, що переміщується, оновлюється або іншим чином змінюється на екрані, не вимагаючи перезавантаження Web-сторінки вручну.

JavaScript спочатку створювався для браузера, але зараз використовується і на інших платформах.

JavaScript можна реалізувати за допомогою операторів JavaScript, які розміщуються в HTML-тегах на Web-сторінці.

<script> ... </script>

Наприклад, код

<script>
    alert('My Web-app!');
</script>

запустить модальне вікно із текстом My Web-app!

JavaScript-код можна розташувати в окремому файлі та підключити до HTML за допомогою атрибута src

<script src="/path/to/myscript.js"></script>

/path/to/myscript.js - абсолютний шлях до скрипту від кореня сайту.

Або вказати відносний шлях від поточної сторінки

<script src="./myscript.js"></script>

Існує можливість включати JavaScript в будь-якому місці HTML-документа. Можна розташувати в розділі head або body.

jQuery - бібліотека JavaScript з відкритим вихідним кодом, яка спрощує взаємодію між документом HTML/CSS або, точніше, об’єктною моделлю документа (DOM) та JavaScript.

Об’єктна модель документа (Document Object Model, DOM) - це крос-платформний і незалежний від мови інтерфейс, який обробляє XML- або HTML-документ як деревоподібну структуру, в якій кожен вузол є об’єктом, що представляє частину документа.

jQuery можна використовувати для пошуку певного елемента HTML у документі HTML з певним ідентифікатором, класом або атрибутом, а потім можемо використовувати jQuery для зміни одного або кількох атрибутів одного й того самого елемента, таких як колір, видимість і т.д.

Також можна використовувати jQuery, щоб зробити Web-сторінку інтерактивною, реагуючи на подію, таку як клік миші.

Основний синтаксис для будь-якої функції jQuery:

$(selector).action()
  • $ призначений для визначення/доступу до jQuery.
  • (selector) призначений для запиту (пошуку) HTML-елементів на сторінці HTML.
  • action() — дія, яка має бути виконана над вибраним елементом (елементами).

Наприклад, код

<script>
    $(function(){
        alert("My Web-app!");
    });
</script>

запустить модальне вікно із текстом My Web-app!

Одним способів застосування бібліотеки jQuery на сторінці HTML є використання мережі доставки контенту (Content Delivery Network, CDN).
