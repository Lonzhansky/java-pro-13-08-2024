
Depth First Search (Обхід у глибину)
--------------------------------------
Обхід із встановленням вершини через змінну.

Реалізація використовує подання списку
суміжності графів.

Мета алгоритму - позначити кожну вершину,
уникаючи циклів (подвійне відвідування вершини).

      0
     / \
    1---2---4
     \  |  /
      \ | /
        3
