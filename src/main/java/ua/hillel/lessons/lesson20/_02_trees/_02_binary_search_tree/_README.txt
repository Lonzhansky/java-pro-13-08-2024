
Бінарне дерево пошуку (Binary search tree, BST)
– структура даних, яка дозволяє отримати відсортований
список чисел.

Називається бінарним деревом, бо кожен вузол дерева
має трохи більше двох дочірніх елементів.
Називається деревом пошуку, тому що його можна
використовувати для пошуку наявності елемента за
час O(logN).

Властивості, які відокремлюють бінарне дерево пошуку від
звичайного бінарного дерева:
- Всі вузли лівого піддерева менше кореневого вузла.
- Усі вузли правого піддерева більше, ніж кореневий вузол.
- Обидва піддерева кожного вузла також є BST, тобто
вони мають дві вищевказані властивості.


Складність бінарного дерева пошуку
-------------------------------------

Time Complexity
Operation	Best Case	Average Case	  Worst Case
Search	    O(logN)	    O(logN)	          O(N)
Insertion	O(logN)	    O(logN)	          O(N)
Deletion	O(logN)	    O(logN)	          O(N)
* N — кількість вузлів в дереві.

Space Complexity
 O(N)
