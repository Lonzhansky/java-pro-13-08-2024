
Class Files
https://docs.oracle.com/javase/8/docs/api/java/nio/file/Files.html

Цей клас складається виключно із статичних методів,
які працюють із файлами, каталогами чи іншими типами
файлів.

У більшості випадків визначені тут методи будуть
делегувати виконання файлових операцій відповідному
провайдеру файлової системи.
