
Interface Path
https://docs.oracle.com/javase/8/docs/api/java/nio/file/Path.html

Interface Path представляє ієрархічний шлях, що складається
з послідовності елементів імені каталогу та файлу,
розділених роздільником.

Екземпляр Java Path являє собою шлях у файловій
системі.
Шлях може вказувати або на файл, або на каталог.
Шлях може бути абсолютним чи відносним.

АБСОЛЮТНИЙ шлях містить повний шлях від кореня файлової
системи до файлу чи каталогу, який він вказує.

ВІДНОСНИЙ шлях містить шлях до файлу або каталогу
щодо деякого іншого шляху.
