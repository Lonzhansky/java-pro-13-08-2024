
Принцип інверсії залежностей
(Dependency Inversion Principle, DIP)
----------------------------------------
Модулі верхніх рівнів не повинні залежати від нижніх
модулів.
Обидва типи модулів мають залежати від абстракцій.
Абстракції не повинні залежати від деталей.
Деталі повинні залежати від абстракцій.
Ми повинні використовувати абстракцію (абстрактні класи
та інтерфейси) замість конкретних реалізацій.
Різні модулі повинні бути автономними та з'єднуватися
один з одним за допомогою абстракції.
