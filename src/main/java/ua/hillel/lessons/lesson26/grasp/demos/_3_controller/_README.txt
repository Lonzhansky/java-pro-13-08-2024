
Controller (Контролер)
--------------------------
Клас, що не відноситься до інтерфейсу користувача та
що відповідає за обробку системних подій.
Використання об'єкта-контролера забезпечує незалежність
між зовнішніми джерелами подій та внутрішніми обробниками
подій, їх типом та поведінкою.

Наочним прикладом може бути використання класу-контролера
у шаблоні MVC.

Контролер відповідає за обробку запитів та вирішує кому
повинен делегувати запити виконання.


Шаблон MVC (Model-View-Controller)
-------------------------------------
Цей шаблон використовується для розділення завдань програми.

Модель (Model) має логіку для оновлення контролера, якщо
його дані змінюються.

Презентація (View) - візуалізація даних, які оброблюються
у моделі.

Контролер (Controller) контролює потік даних в об'єкт
моделі та оновлює презентацію при кожній зміні даних.
Він поділяє презентацію та модель.
