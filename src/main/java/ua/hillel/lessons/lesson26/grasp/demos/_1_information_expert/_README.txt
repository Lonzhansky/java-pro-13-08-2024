
Information Expert (Інформаційний Експерт)
----------------------------------------------
Інформація має оброблятися ТАМ, ДЕ вона міститься.

Інформаційні експерти — КЛАСИ, які мають інформацію, що
необхідна для виконання функціональності програмної системи.

Відповідальність за функціонал має бути призначена тому,
хто володіє максимумом необхідної інформації для виконання,
тобто інформаційному експерту.

Застосування шаблону інформаційний експерт підвищує зв'язність
модулів і не суперечить властивості інкапсуляції.

Є ризик "спагетті-коду" якщо не враховувати цей шаблон.


Демо
------
Є найпростіша система класів: Order (замовлення), що містить
список OrderItem-ів (одиниць замовлення), елементи якого,
у свою чергу, містять Product (товар) та його кількість,
а товар може містити, наприклад, ціну, назву та ін.
