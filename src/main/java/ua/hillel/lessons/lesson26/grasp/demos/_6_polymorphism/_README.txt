
Polymorphism (Поліморфізм)
----------------------------
Поліморфізм дозволяє реалізовувати ОДНОІМЕННІ публічні
МЕТОДИ, дозволяючи різним класам виконувати різні дії
при одному і тому ж виклику.

Поліморфізм дозволяє обробляти АЛЬТЕРНАТИВНІ ВАРІАНТИ
поведінки з урахуванням типу. При цьому альтернативні
реалізації наводяться до узагальненого інтерфейсу.
