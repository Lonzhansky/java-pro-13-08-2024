
ПОГАНО.

Клас містить два методи.
Один метод полягає у додаванні двох чисел
(аріфметична операція), а інший метод фокусується
на голосних та приголосних (пов'язаних з алфавітом).

Однак ці два методи об'єднані в один клас,
але немає жодного зв'язку між гласними чи приголосними
та числами.

Це приклад відсутності зчеплення (спеціалізації класу).
