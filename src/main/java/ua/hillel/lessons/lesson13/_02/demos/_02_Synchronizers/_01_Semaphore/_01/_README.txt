
Можемо використовувати семафор, щоб заблокувати доступ до ресурсу,
кожен потік, який хоче використовувати цей ресурс, має спочатку
викликати метод acquire(), перш ніж отримати доступ до ресурсу для
отримання блокування

Коли потік закінчить роботу з ресурсом, він має викликати функцію
release(), щоб зняти блокування.
