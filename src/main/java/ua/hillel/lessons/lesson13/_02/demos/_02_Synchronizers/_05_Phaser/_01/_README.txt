
ПРИКЛАД. Є два виконувані завдання, які будуть
виконуватися як дві окремі фази.

Перше завдання (FirstTask) виконується трьома потоками,
можна бачити, що 3 сторони зареєстровані з використанням
методу bulkRegister().

Для другої виконуваної задачі (SecondTask) викликається метод
реєстрації у виконуваному класі.
