
Class Exchanger (Обмінник)
https://docs.oracle.com/javase/8/docs/api/java/util/concurrent/Exchanger.html

У ТОЧЦІ, в якій два потоки можуть ОБМІНЮВАТИСЯ інформацією,
Exchanger діє як точка синхронізації.
Доки його аналог(обмінник) не надасть свою інформацію, потоки
блокуються.
Таку ж поведінку можна спостерігати з обох боків.

Exchanger в Java - ЗАСІБ СИНХРОНІЗАЦІЇ, який допомагає
обмінюватись даними між двома потоками. Коли два потоки
викликають метод exchange(), об'єкти, надані цими двома
потоками, обмінюються.

Exchanger можна використовувати у сценарії «виробник-споживач»
(Producer-Consumer), де потік-виробник створює буфер, який
можна обміняти на порожній буфер із потоку-споживача.
