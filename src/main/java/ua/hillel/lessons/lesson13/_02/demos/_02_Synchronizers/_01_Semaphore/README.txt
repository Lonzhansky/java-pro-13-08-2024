
Class Semaphore
https://docs.oracle.com/javase/8/docs/api/java/util/concurrent/Semaphore.html

Class Semaphore КЕРУЄ доступом до загального ресурсу за допомогою
ЛІЧИЛЬНИКА.
Якщо лічильник БІЛЬШЕ НУЛЯ, доступ дозволено.
Якщо він рівний нулю, то доступ заборонено.
Лічильник рахує дозволи, що дозволяють доступ до загального ресурсу.
Таким чином, для доступу до ресурсу потік повинен отримати дозвіл від
семафору.

Class Semaphore - це ЗАСІБ синхронізації, який можна використовувати
для взаємодії між потоками або обмеження доступу до ресурсу.


Робота Semaphore
-------------------
Щоб використати семафор, потік, який
хоче отримати доступ до спільного ресурсу, намагається
отримати дозвіл.

Якщо лічильник семафора більший за нуль, то потік отримує
дозвіл, що призводить до зменшення лічильника семафору.

В іншому випадку потік буде заблокований доти,
доки не буде отримано дозвіл.

Коли потоку більше не потрібний доступ до спільного ресурсу,
він звільняє дозвіл, що призводить до збільшення
лічильника семафору.

Якщо є інший потік, що очікує на дозвіл, то цей
потік отримає дозвіл у цей час.
