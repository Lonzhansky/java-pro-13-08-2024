
Class CountDownLatch
(Клямка зворотного відліку)
https://docs.oracle.com/javase/8/docs/api/java/util/concurrent/CountDownLatch.html

У CountdownLatch один або кілька потоків налаштовані на
ОЧІКУВАННЯ завершення набору операцій в інших потоках.
Ініціалізується кількість потоків для встановлення очікування.
Поки лічильник не досягне нуля, метод очікування блокується.

CountDownLatch в Java - це ЗАСІБ СИНХРОНІЗАЦІЇ, який
дозволяє одному або декільком потокам очікувати завершення
набору операцій, які виконуються в інших потоках.


Робота CountDownLatch
-----------------------
CountDownLatch ініціалізується із заданим лічильником,
потім є два методи await() і countDown(), які
блокують та звільняють потоки.

await() — змушує поточний потік чекати, поки засувка
не дійде до нуля.

countDown() - зменшує лічильник засувки, звільняючи все
очікують потоки, якщо лічильник досягає нуля.

CountDownLatch - закриває засувку, так що поточний
потік блокується при виклику методу await і чекає, поки
зворотній відлік не досягне нуля від заданого лічильника.
Коли лічильник досягає нуля, засувка відкривається для
продовження потоку.
