
Class StringBuilder
https://docs.oracle.com/javase/8/docs/api/java/lang/StringBuilder.html

Class StringBuilder використовується для створення змінного
(mutable) рядка.
Class StringBuilder аналогічний класу Class StringBuffer, крім
те, що він несинхронізований. Він доступний з JDK 1.5.

Class StringBuilder не синхронізовано, тобто не потокобезпечний.
Це означає, що два потоки можуть одночасно викликати методи
Class StringBuilder.
