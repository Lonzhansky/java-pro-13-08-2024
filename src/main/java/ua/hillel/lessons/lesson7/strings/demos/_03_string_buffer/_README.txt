
Class StringBuffer
https://docs.oracle.com/javase/8/docs/api/java/lang/StringBuffer.html

Class StringBuffer використовується для створення змінного
(mutable) рядка.
Class StringBuffer такий же, як Class String, за винятком
того, що він змінюється.

Class StringBuffer є потокобезпечним, тобто кілька потоків
не можуть отримати доступ до нього одночасно.
Так що це безпечно і веде до порядку.
