
Class StringJoiner
https://docs.oracle.com/javase/8/docs/api/java/util/StringJoiner.html

Class StringJoiner (з'єднувач) використовується для побудови послідовності
символів, розділених роздільником і необов’язково починаючи з
наданого префікса та закінчуючи наданим суфіксом.

Тобто, Class StringJoiner  можна використовувати для об’єднання
рядків за допомогою розділювача, префікса та суфікса.
