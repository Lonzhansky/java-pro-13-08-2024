
Class String
https://docs.oracle.com/javase/8/docs/api/java/lang/String.html

String в Java - це ОБ'ЄКТ, що представляє послідовність
символів.
Усі літерали String, наприклад, «Orange», реалізовані
як об'єкти класу String.

ЛІТЕРАЛ - КОНСТАНТНЕ значення, яке може бути надано
змінної.

У Java рядкові об'єкти НЕЗМІНЮВАНІ (immutable).
Після створення рядкового об'єкта його дані чи стан
не можна змінити, але створюється новий рядковий об'єкт.
