
Порівняння рядків
-------------------

Якщо потрібно ПОРІВНЯТИ два рядки в Java
або частину двох рядків на основі вмісту
цих рядків, можна зробити це, використовуючи
відповідні методи класу String.

Методи equals() або equalsIgnoreCase(),
якщо не хочемо враховувати регістр.

Методи compareTo() або compareToIgnoreCase(),
якщо не хочемо враховувати регістр.

Для порівняння частини рядка можемо використати
методи startsWith() та endsWith().

Щоб порівняти область одного рядка із зазначеною
областю іншого рядка, можемо використовувати метод
regionMatches().


equals() vs == в Java
-----------------------

Використовуються для порівняння об'єктів з метою
перевірки рівності.

Метод equals() порівняння вмісту.

== перевіряє, чи вказують обидва об'єкти на
одне й те місце у пам'яті.

equals() – це МЕТОД, а == – це ОПЕРАТОР.

Оператор == для порівняння посилань (порівняння АДРЕС),
метод equals() для порівняння ЗМІСТУ.

Оператор == перевіряє, чи обидва об'єкти вказують на одне
і те ж МІСЦЕ в пам'яті, тоді як equals() оцінює
рівність ЗНАЧЕНЬ в об'єктах String.

Якщо клас не перевизначає метод equals(), то
за замовчуванням він використовує метод equals(Object o)
найближчого батьківського класу, який перевизначив цей метод.
