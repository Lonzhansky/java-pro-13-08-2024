
Валідація паролю
___________________

Пароль вважається дійсним,
якщо виконуються всі такі обмеження:
- Містить щонайменше 8 символів та не більше 20 символів.
- Містить щонайменше одну цифру.
- Містить щонайменше одну літеру верхнього регістру.
- Містить щонайменше одну літеру нижнього регістру.
- Містить щонайменше один спеціальний символ,
   у тому числі !@#$%&*()-+=^.
- Не містить пробілів.

Регулярний вираз для перевірки правильності
пароля:

^(?=.*[0-9])(?=.*[a-z])(?=.*[A-Z])(?=.*[@#$%^&-+=()])(?=\\S+$).{8, 20}$

^                       початковий символ рядка.

(?=.*[0-9])             цифра має зустрічатися хоча б один раз.

(?=.*[a-z])             літери нижнього регістру, повинні
                        зустрічатися хоча б один раз.

(?=.*[A-Z])             літери верхнього регістру, повинні
                        зустрічатися хоча б один раз.

(?=.*[@#$%^&-+=()]      спеціальний символ, який має
                        зустрічатися хоча б один раз.

(?=\\S+$)               пробіли не допускаються у всьому рядку.

.{8, 20}                не менше 8 та не більше 20 символів.

$                       кінец рядка.
