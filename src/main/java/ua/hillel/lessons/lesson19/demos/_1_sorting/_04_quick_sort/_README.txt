
Quick sort (Швидке сортування)

https://en.wikipedia.org/wiki/Quicksort

Це високоефективний алгоритм сортування, заснований
на розбиття масиву даних на більш дрібні масиви.

Великий масив поділений на два масиви, один з яких
містить менше значення, ніж зазначене значення, а інший
варіант містить значення, що перевищує опорне значення.
