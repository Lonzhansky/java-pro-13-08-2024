
Selection sort (Сортування вибором)

https://en.wikipedia.org/wiki/Selection_sort

Алгоритм, який знаходить найменший елемент у несортованій частині
масиву та обмінює його з першим елементом несортованої частини.

Від найменшого до найбільшого.
Починається з пошуку найменшого елемента в масиві та
замінення його першим.

Потім знаходиться другий найменший елемент та заміняється
другим, і так доти, поки весь масив не відсортується.
