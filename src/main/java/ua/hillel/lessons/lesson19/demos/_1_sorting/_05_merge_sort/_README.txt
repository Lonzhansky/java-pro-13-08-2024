
Merge sort (Сортування злиттям)

https://en.wikipedia.org/wiki/Merge_sort

Сортування злиттям — один із найефективніших алгоритмів
сортування, що працює за принципом «розділяй та володарюй».

* Алгоритм принципу «розділяй та володарюй» розбиває проблему
на підзавдання, вирішує одне підзавдання та об'єднує рішення
разом, щоб отримати остаточне рішення.

Сортування злиттям неодноразово розбиває масив на кілька
частин, доки кожна частина не буде складатися з одного
елемента і об'єднує ці частини таким чином, щоб отримати
відсортований масив.
