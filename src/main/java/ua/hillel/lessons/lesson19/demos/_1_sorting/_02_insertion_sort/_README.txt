
Insertion sort (Сортування вставками)

https://en.wikipedia.org/wiki/Insertion_sort

На кожному кроці алгоритму вибирається один із елементів
вхідних даних (масиву) та вставляється на потрібну позицію
у вже відсортованій частині масиву поки весь масив вхідних
даних не буде відсортований.

Алгоритм переглядає дані у вигляді двох частин.
Ліва частина - частина відсортованих елементів, права які потрібно
відсортувати.
На кожній ітерації один елемент з правої частини береться
і додається у ліву частину так, що ліва частина залишається
відсортованою.
