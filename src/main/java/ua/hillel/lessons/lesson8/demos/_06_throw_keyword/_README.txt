
Ключове слово throw
----------------------
Використовується для створення та явного викидання одного винятку.
Вказується об'єкт того типу винятку, який має бути згенерований.
Викидання винятку може бути з методу або будь-якого блоку коду.
Як перевіряємі, так і неперевіряємі винятки можуть бути створені
за допомогою ключового слова throw.

Виняток містить повідомлення з описом помилки.
Такі винятки можуть бути пов'язані, наприклад, з валідацією
вхідних даних щодо правильності введення, відповідності
певному критерію, тощо.

Коли виняток створюється за допомогою ключового слова throw,
потік виконання програми припиняється, а керування передається
найближчому охоплюючому блоку try-catch, який відповідає типу
викинутого винятку. Якщо такого збігу не знайдено, обробник винятків
за замовчуванням завершує програму.

Ключове слово throw корисне для створення кастомного винятку.
