
Ключове слово throws
------------------------
Використовується для оголошення винятку в методі.
Повідомляє розробнику, що може статися виняток.
Таким чином, для розробника необхідно надати код
обробки винятків, щоб можна було підтримувати
нормальний потік виконання програми.

Обробка винятків переважно використовується для обробки
винятків, що перевіряються.

Якщо виникає якийсь неперевіряємий виняток,
такий як ArithmeticException, розробник винний у тому,
що не перевіряє код перед його використанням.

При використанні ключового слова throws, перевіряємий
виняток може бути поширеним (перенаправлено у стек викликів).
Це надає інформацію про виняток стороні, що викликає.
