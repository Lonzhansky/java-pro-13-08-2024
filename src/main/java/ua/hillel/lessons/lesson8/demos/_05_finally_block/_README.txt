
Блок finally
--------------
Блок finally в Java - блок, який використовується
для виконання важливого коду, такого як закриття
з'єднання, потоку і т.д.

Блок finally в Java завжди виконується незалежно
від того, чи оброблено виняток чи ні.

Блок finally слідує за блоком try-catch.

Для кожного блоку try може бути нуль або більше блоків catch,
але тільки один блок нарешті.
