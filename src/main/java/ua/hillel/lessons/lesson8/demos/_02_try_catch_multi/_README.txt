
Множинний блок catch
---------------------------
За блоком try може слідувати один або
кілька блоків catch.

Кожен блок catch повинен містити окремий
обробник винятків.

Якщо потрібно виконувати різні завдання, у разі виникнення
різних винятків, використовуємо множинний блок catch.

Одночасно виникає лише один виняток,
і одночасно виконується лише один блок catch.

Усі блоки перехоплення мають бути упорядковані від
найбільш конкретних до найбільш загальних, тобто перехоплення
для ArithmeticException повинно передувати перехопленню
для Exception.
