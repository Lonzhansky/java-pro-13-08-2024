
ТРАСУВАННЯ СТЕКУ (stack trace) Java відображається у разі
виникнення помилки чи винятку.

Трасування стеку, ЗВОРОТНЄ ТРАСУВАННЯ, складається
з набору записів стеку, в яких зберігається РУХ
програми під час її виконання.

Трасування стеку в Java - СПИСОК всіх ВИКЛИКІВ методів,
які програма виконувала до виникнення помилки.
Воно генерується автоматично, коли коді виникає виняток.

Трасування стеку є МАСИВ ФРЕЙМІВ/кадрів стеку.
Кадри стеку є РОЗВИТОК додатка під час
виконання програми.
