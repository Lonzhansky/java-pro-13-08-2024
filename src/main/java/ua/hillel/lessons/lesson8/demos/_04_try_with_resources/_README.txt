
Оператор try-with-resources - це оператор try,
який декларує один або кілька ресурсів.

Ресурс - об'єкт, який повинен бути закритий після того, як
як програма закінчить роботу з ним.

Оператор try-with-resources гарантує, що кожен ресурс буде
закрито наприкінці оператора.
Як ресурс може використовуватися будь-який об'єкт, що реалізує
java.lang.AutoCloseable, включаючи всі об'єкти, які
реалізують java.io.Closeable.

Таким чином, не потрібно додавати додатковий блок finally,
для простої передачі операторів ресурсів, що закривають.
Ресурси буде закрито, як тільки буде виконано блок try-catch.
