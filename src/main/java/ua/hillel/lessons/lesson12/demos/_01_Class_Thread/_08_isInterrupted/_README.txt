
Метод isInterrupted() перевіряє, чи не було перервано потік.
Цей метод не впливає на перерваний стан потоку.
Переривання потоку, яке ігнорується через те, що потік не був активним
під час переривання, буде відображено цим методом, що повертає false.

interrupted() vs isInterrupted()
----------------------------------
Метод interrupted() перевіряє, перервано чи ні потік, і повертає логічне
значення, але після цього змінює своє значення з true на false, якщо значення
вже є true.

Метод isInterrupted() перевіряє значення та ніколи його не змінює, тобто якщо
статус має значення true, він залишиться true, але interrupted() зміниться
з true на false.
