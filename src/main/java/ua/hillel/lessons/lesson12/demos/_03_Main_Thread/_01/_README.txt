
Як керувати основним потоком
---------------------------------
Основний потік створюється автоматично під час
запуску програми.

Щоб керувати ним, ми маємо отримати посилання на нього.
Це можна зробити, викликавши метод currentThread(), який
визначено в Class Thread. Цей метод повертає посилання
на потік, у якому він викликається.

Пріоритет основного потоку за замовчуванням дорівнює 5,
а для решти користувацьких потоків пріоритет буде
успадкований від батьківського до дочірнього.
