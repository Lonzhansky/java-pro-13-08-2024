
Метод interrupted() перевіряє, чи не було перервано поточний потік.
Перерваний стан потоку очищається цим методом.
Іншими словами, якби цей метод викликався двічі поспіль, другий виклик
повернув би false (якщо б поточний потік не був перерваний знову після
того, як перший виклик очистив свій статус перерваного і до того, як
другий виклик перевірив його).
Повертає true, якщо поточний потік було перервано, інакше false.
