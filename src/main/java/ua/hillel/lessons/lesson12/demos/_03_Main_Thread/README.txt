
Основний (Main) потік у Java
-------------------------------

Багатопотокова програма складається з двох або більше частин,
які можуть виконуватись одночасно.

Кожна частина такої програми називається потоком, і кожен потік
визначає окремий шлях виконання.

Коли програма Java запускається, відразу починає виконуватися
один потік.
Зазвичай його називають основним (main) потоком програми, тому що він
виконується під час запуску програми.

Метод main() є точкою входу для виконання програми.
Таким чином, коли JVM починає виконання програми, вона створює
потік для її запуску і цей потік називається основним потоком.


Властивості, пов'язані з основним потоком:
- Це потік, з якого будуть створені інші дочірні потоки.
- Часто це має бути останній потік для завершення виконання
програми, оскільки він виконує різні дії із завершення роботи.
