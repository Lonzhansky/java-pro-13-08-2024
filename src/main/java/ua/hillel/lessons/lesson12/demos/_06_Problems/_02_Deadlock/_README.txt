
Взаємоблокування (Deadlock) є проблемою багатопоточності.

Взаємоблокування може виникнути у ситуації, коли потік очікує блокування
об'єкта, отриманого іншим потоком, а другий потік очікує блокування
об'єкта, отриманого першим потоком.

Оскільки обидва потоки чекають один на одного, щоб зняти блокування,
таке стан називається взаємоблокуванням.
