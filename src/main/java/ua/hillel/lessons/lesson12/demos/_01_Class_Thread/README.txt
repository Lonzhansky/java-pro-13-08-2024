
Class Thread надає певні методи для маніпулювання потоками.
Для розуміння процесу роботи з потоками, розглянемо декілька
з них.
https://docs.oracle.com/en/java/javase/17/docs/api/java.base/java/lang/Thread.html

УВАГА. Результат виконання програм може залежати від певних
факторів: операційна система, налаштування JVM на конкретній
локальній машині.  Якщо є кілька потоків, те який потік отримає
шанс виконання першим, вирішуватиме Планувальник потоків
(Thread Scheduler). Планувальник потоків є частиною JVM.
Поведінка планувальника потоків залежить від постачальника JDK,
і тому ми не можемо очікувати гарантовано однакового результату
кожного разу при виконані програми.
