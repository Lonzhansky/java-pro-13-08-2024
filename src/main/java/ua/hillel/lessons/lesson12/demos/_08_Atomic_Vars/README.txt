
Атомарні змінні (Atomic Variables)
-------------------------------------

Java в пакеті java.util.concurrent.atomic пропонує набір класів,
які підтримують безблокове потокобезпечне програмування для окремих
змінних.

https://docs.oracle.com/javase/8/docs/api/java/util/concurrent/atomic/package-summary.html

Використання класів, які містяться в цьому пакеті, гарантує атомарність
операцій для int, long, boolean значень, а також посилань на об'єкти та
масиви без використання явної синхронізації або блокування.

Атомарні класи використовують інструкції порівняння та заміни -
CAS (Compare and Swap) – для забезпечення цілісності даних з використанням
неблокуючих алгоритмів.
Тому ці класи вважаються швидшими, ніж блокування, коли один потік отримує
блокування об'єкта, інші потоки блокуються.
