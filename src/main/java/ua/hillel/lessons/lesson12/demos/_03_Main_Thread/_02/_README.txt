
Зв'язок між методом main() та основним потоком Java
-----------------------------------------------------
Кожна програма JVM створює основний потік.
Головний потік спочатку перевіряє існування методу main(),
а потім ініціалізує його клас.

Починаючи з JDK 6, метод main() є обов'язковим в окремому
Java-додатку.

Ми можемо створити взаємоблокування, тобто, просто
використовуючи один потік. Наприклад, за допомогою
основного потоку.
